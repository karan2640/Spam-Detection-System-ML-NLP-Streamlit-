📩 Spam Detection System (ML + NLP + Streamlit)

An end-to-end Machine Learning project that classifies SMS/Email messages as Spam 🚨 or Not Spam ✅ using Natural Language Processing (NLP) techniques and an interactive web app built with Streamlit.

🚀 Features
🔍 Real-time spam detection
🧠 NLP preprocessing (tokenization, stopword removal, stemming)
📊 TF-IDF vectorization
🤖 Machine Learning model (Multinomial Naive Bayes)
📈 Confidence score with visualization
🕒 Prediction history tracking
🎨 Clean and interactive UI
